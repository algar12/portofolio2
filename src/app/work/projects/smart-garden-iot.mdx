## Ringkasan

Sistem otomasi taman pintar berbasis IoT yang mengombinasikan backend Laravel dengan embedded sensors untuk menciptakan solusi perawatan tanaman yang intelligent. Sistem memantau kondisi tanah dan mengotomasi irigasi berdasarkan data real-time, membuat perawatan tanaman menjadi mudah dan efisien.

## Fungsionalitas Inti

### Irigasi Otomatis
- **Smart Watering**: Kontrol pompa otomatis berdasarkan level kelembaban tanah
- **Konfigurasi Threshold**: Threshold kelembaban yang dapat disesuaikan untuk berbagai jenis tanaman
- **Manajemen Jadwal**: Jadwal penyiraman berbasis waktu dengan kemampuan override
- **Konservasi Air**: Mencegah over-watering melalui monitoring cerdas

### Monitoring Real-Time
- **Kelembaban Tanah**: Tracking kontinu level humidity tanah
- **Data Lingkungan**: Integrasi sensor suhu dan cahaya opsional
- **Tracking Historis**: Logging data untuk analisis pola
- **Sistem Alert**: Notifikasi untuk kondisi kritis (terlalu kering/terlalu basah)

## Arsitektur Teknis

### Backend (Laravel)
- **API Endpoints**: RESTful API untuk ingesting dan retrieval data sensor
- **Database**: MySQL untuk storing pembacaan sensor dan konfigurasi
- **Dashboard**: Interface web untuk monitoring dan kontrol
- **Manajemen User**: Support multi-user dengan role-based access

### Layer IoT
- **Mikrokontroler**: ESP8266 atau board WiFi-enabled serupa
- **Sensors**:
  - Sensor kelembaban tanah
  - DHT22 untuk suhu dan humidity (opsional)
  - Sensor cahaya untuk tracking pertumbuhan optimal
- **Actuators**: Modul relay untuk kontrol pompa
- **Protokol**: MQTT untuk komunikasi low-latency yang reliable

### Alur Komunikasi
1. **Pengumpulan Data**: Sensor membaca parameter lingkungan
2. **Transmisi**: ESP8266 mengirim data ke Laravel API via MQTT/HTTP
3. **Processing**: Laravel memproses data dan menentukan action
4. **Otomasi**: Sistem trigger irigasi jika diperlukan
5. **Logging**: Semua event dan pembacaan disimpan untuk analisis
